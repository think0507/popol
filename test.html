System Font + Colorful Theme

이 예시는 System Font Stack을 사용하면서, 보라톤 계열 색상으로 제목과 버튼, 링크 등에 포인트를 준 스타일입니다.

섹션 부제목 (H2)

H2는 약간 짙은 보라색(#7c3aed)을 사용했습니다. 본문의 텍스트는 #444로 설정해 너무 어둡지 않게 조정했습니다.

  • 두 번째 리스트 항목
  • 세 번째 리스트 항목

소제목 (H3)

H3는 좀 더 밝은 보라(#9b75f7)를 적용했습니다. 링크도 마우스오버 시 더 밝은 색(#9b75f7)로 변합니다.

다른 섹션 (H2)

버튼 역시 기본 보라(#5f2eea)에서 호버 시 보라톤(#7c3aed)으로 바뀝니다.

Tip: 색상이 마음에 안 드시면 #5f2eea, #7c3aed, #9b75f7 대신 다른 계열의 컬러를 사용해보세요!
